## 模块一：右侧买卖改为 Tab 切换

### 改动说明

当前是买入、卖出两列并排，改为 Tab 切换单列展示：

```
改前：
[ 买入 BTC ]  [ 卖出 BTC ]   ← 两列并排

改后：
[  买入  ] [  卖出  ]        ← Tab 切换
价格(USDT)
[        0.00       USDT ]
数量(BTC)
[     0.000000      BTC  ]
25%  50%  75%  100%
可用  --  USDT
杠杆  [ 10x ▼ ]
[ 买入 BTC  ]   ← 绿色按钮（买入状态）
               ← 卖出时显示红色「卖出 BTC」
```

### 行为逻辑

- 默认选中「买入」Tab
- 切换 Tab 时：
  - 按钮颜色切换（买入绿色 `#26a69a`，卖出红色 `#ef5350`）
  - 表单状态独立（价格、数量、订单类型各自保存，切换时不重置）
- Tab 选中项文字加粗，下方加彩色下划线指示

### 好处

右侧面板宽度不变，但内容更聚焦，K 线图可以横向扩展占用原来右侧一列的空间。

## 模块二：底部持仓 / 委托 / 历史成交 Tab 面板

### Tab 一：当前持仓

列表表头：
```
合约    方向    数量    开仓均价    标记价格    未实现盈亏    收益率    强平价格    操作
```

空状态：居中显示「暂无持仓」灰色文字

模拟数据（用于展示效果，固定写死）：
```ts
const mockPositions = [
  {
    symbol: 'BTCUSDT',
    side: 'long',           // 'long' | 'short'
    size: 0.05,
    entryPrice: 71200.00,
    markPrice: 71540.00,    // 从现有 store 取实时价格
    leverage: 10,
  }
]
```

未实现盈亏计算：
```ts
// 多仓
pnl = (markPrice - entryPrice) * size
// 空仓
pnl = (entryPrice - markPrice) * size

收益率 = pnl / (entryPrice * size / leverage) * 100
```

盈亏颜色：正数绿色，负数红色。

操作列：「市价平仓」按钮（点击后从 mockPositions 移除该条，模拟平仓）

interface PositionStore {
